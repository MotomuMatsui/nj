## Installation
### 0. Requirements

- [GNU GCC compiler](https://gcc.gnu.org/) (5.0+) is required to compile `nj`

### 1. Compile from source code:

````
    $ git clone https://github.com/MotomuMatsui/nj
    $ cd nj
    $ make
````
